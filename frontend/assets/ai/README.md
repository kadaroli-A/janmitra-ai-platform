# JanMitra AI Avatar

Place the `janmitra-bot.png` image file in this directory.

The image should be:
- A friendly AI robot avatar
- Transparent background (PNG format)
- Recommended size: 512x512px or larger
- The avatar will be displayed at various sizes throughout the app

## Usage Locations

1. **Header Logo** - 40x40px circular
2. **Voice Input Page** - 120x120px with floating animation
3. **Analysis Page** - 150x150px with floating and pulse animation
4. **Scheme Details** - 40x40px when speaking

## Fallback

If the image is not found, the app will display a 🤖 emoji as a fallback.
